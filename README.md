# Web Scraping Projects Collection 🕸️

A repository containing various web scraping scripts developed to extract, clean, and store data from multiple online sources. This collection ranges from basic scraping techniques to more complex data extraction tasks.

## 📂 Directory Structure

| Folder | Description |
| :--- | :--- |
| **A-Z_Animals_data** | Scrapes comprehensive data about animal species, habitats, and characteristics. |
| **Movie_boxoffice_data** | Extracts financial performance and ranking data for global box office hits. |
| **Oscar Winning Films** | Gathers historical records of Academy Award-winning movies and nominees. |
| **simple country** | A lightweight scraper for extracting country names, capitals, and basic geographic stats. |
| **team** | Scripts designed to scrape sports team rosters or organizational data. |
| **basic** | Fundamentals of web scraping, including HTML parsing and simple requests. |

---

## 🛠️ Tech Stack & Tools

These projects are built using **Python** and the following libraries:

* **BeautifulSoup4**: For parsing HTML and XML documents.
* **Requests**: For sending HTTP requests to target websites.
* **Pandas**: For data manipulation, cleaning, and exporting to CSV/Excel.
* **LXML**: High-performance HTML/XML parsing (used as a backend for BeautifulSoup).
